# Online-food--delivery-system
Online Food Delivery System in C++ using the Win32 API. Features include secure user login, interactive food menu, order placement, quantity selection, automatic bill calculation, rider assignment, order tracking, and order history. Demonstrates GUI development, event-driven programming, and data structure implementation.
The Online Food Delivery System is a desktop application developed in C++ using the Win32 API. It simulates the workflow of a modern food delivery service by allowing users to log in, browse a food menu, place orders, calculate bills, assign delivery riders, and track order status. The system provides an interactive graphical user interface (GUI) and demonstrates the implementation of core programming concepts such as data structures, event-driven programming, and file-independent order management.

Key Features
Secure user login authentication.
Interactive graphical user interface (GUI) using the Win32 API.
Food menu with multiple categories (Pizza, Burger, Pasta, Drinks, and Desserts).
Quantity selection for each order.
Automatic bill calculation based on selected items.
Optional rider assignment for food delivery.
Order history management.
Real-time order tracking with date and time.
Displays the day and timestamp for every order.
Technologies Used
Programming Language: C++
Framework/API: Win32 API
Data Structure: Queue (for order management)
Concepts: Object-oriented programming, GUI development, event handling, and basic data management.
Project Objective

The objective of this project is to develop a user-friendly food ordering application that automates the food delivery process. It provides customers with an easy way to place and track orders while demonstrating practical C++ programming skills, GUI development, and data structure implementation.
